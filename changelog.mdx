## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration features allowing multiple users to edit documents simultaneously
- Introduced advanced search with filters for content type, author, and date ranges
- Implemented webhook notifications for project and page changes
- Added support for custom page templates with drag-and-drop editor

## Improvements

- Enhanced API performance with response times reduced by 40%
- Improved mobile responsiveness across all documentation pages
- Updated user interface with modern design elements and better accessibility
- Optimized file upload process for larger documentation assets

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking", "feature"]}>

## New Features

- Launched complete API redesign with RESTful endpoints and comprehensive SDKs
- Introduced team management system with role-based permissions
- Added project analytics dashboard showing usage metrics and engagement data
- Implemented version control system for all documentation changes

## Breaking Changes

- API endpoints now require authentication headers for all requests
- Project visibility settings changed from public/private to granular permission levels
- Webhook payload structure updated to include additional metadata fields

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added Git repository integration for automatic content synchronization
- Introduced markdown import functionality with format conversion
- Implemented user notification system for mentions and updates
- Added bulk operations for managing multiple pages and projects

## Bug Fixes

- Fixed search indexing issues that caused some pages to be unsearchable
- Resolved permission inheritance problems in nested project folders
- Corrected markdown rendering inconsistencies in code blocks
- Fixed webhook delivery failures under high load conditions

</Update>
