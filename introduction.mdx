## Key Features

You work with a robust platform designed for modern documentation needs. The system supports real-time collaboration, version control, and seamless integration with popular development tools.

<Tabs>
  <Tab title="Collaboration" icon="users">
    Invite team members to collaborate on documentation projects in real-time. You assign roles and permissions to control access levels, ensuring secure and efficient teamwork.
  </Tab>
  <Tab title="Version Control" icon="git-branch">
    Track changes to your documentation with built-in version control. You review edit history, revert to previous versions, and maintain audit trails for compliance purposes.
  </Tab>
  <Tab title="Integrations" icon="plug">
    Connect WIZIUM with your existing tools. You import content from Git repositories, sync with project management platforms, and export documentation in multiple formats.
  </Tab>
</Tabs>

## Getting Started

You begin by creating your WIZIUM account and setting up your first documentation project. Follow these steps to establish your documentation workspace.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your WIZIUM account using your email address or through SSO integration with your organization's identity provider.
  </Step>
  <Step title="Create Project" icon="plus">
    Set up your first documentation project. You define the project name, description, and initial team members.
  </Step>
  <Step title="Import Content" icon="upload">
    Import existing documentation from various sources. You upload Markdown files, connect Git repositories, or migrate from other platforms.
  </Step>
  <Step title="Customize Settings" icon="settings">
    Configure project settings to match your workflow. You set up custom templates, define access controls, and enable integrations.
  </Step>
</Steps>

<Callout kind="info">
  Pro tip: Start with a small pilot project to familiarize yourself with WIZIUM's features before scaling up to larger documentation initiatives.
</Callout>

## Support and Resources

You access comprehensive support resources to maximize your WIZIUM experience. The platform includes detailed guides, video tutorials, and a responsive support team.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
const wizium = require('wizium-sdk');

wizium.authenticate({
  apiKey: 'your-api-key',
  projectId: 'your-project-id'
});

wizium.projects.list().then(projects => {
  console.log('Your projects:', projects);
});
```
  ```python
import wizium_sdk

client = wizium_sdk.Client(
    api_key='your-api-key',
    project_id='your-project-id'
)

projects = client.projects.list()
print(f"Your projects: {projects}")
```
</CodeGroup>
