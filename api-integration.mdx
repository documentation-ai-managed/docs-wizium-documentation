## Authentication

You authenticate API requests using API keys or OAuth tokens. The authentication system ensures secure access to your resources.

<Steps>
  <Step title="Generate API Key" icon="key">
    Access your account settings and create an API key. You copy the key for use in requests.
  </Step>
  <Step title="Include in Headers" icon="header">
    Add the Authorization header to all API requests. You use Bearer token format.
  </Step>
  <Step title="Handle Tokens" icon="refresh-cw">
    Rotate API keys regularly and revoke compromised tokens. You monitor usage.
  </Step>
</Steps>

<CodeGroup tabs="cURL,JavaScript,Python">
  ```bash
# Get API key from dashboard
curl -X GET https://api.wizium.com/projects \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```
  ```javascript
const wizium = require('wizium-sdk');

wizium.authenticate({
  apiKey: 'your-api-key'
});

// API calls now authenticated
wizium.projects.list();
```
  ```python
import wizium_sdk

client = wizium_sdk.Client(api_key='your-api-key')

# Authenticated requests
projects = client.projects.list()
```
</CodeGroup>

<Callout kind="danger">
  Never expose API keys in client-side code or public repositories. Use environment variables for secure storage.
</Callout>

## Content Management

You manipulate documentation content using the Content API. Create pages, update content, and manage versions.

<Steps>
  <Step title="Create Pages" icon="file-plus">
    Add new documentation pages to projects. You specify content, metadata, and folder location.
  </Step>
  <Step title="Update Content" icon="edit">
    Modify existing page content. You preserve version history and track changes.
  </Step>
  <Step title="Manage Assets" icon="image">
    Upload and link images and files. You organize media within projects.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
// Create a documentation page
const page = await wizium.pages.create({
  projectId: 'project-123',
  title: 'Getting Started',
  content: '# Welcome\n\nThis guide helps you get started.',
  folderId: 'folder-456',
  tags: ['tutorial', 'beginner']
});

console.log('Page created:', page.id);
```
  ```python
# Update page content
client.pages.update(
    page_id='page-789',
    content='# Updated Content\n\nNew information here.',
    message='Updated getting started guide'
)
```
</CodeGroup>

<Callout kind="info">
  Use meaningful commit messages when updating content to maintain clear version history.
</Callout>
